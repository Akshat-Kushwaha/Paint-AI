# Paint-AI
A Python-based automation tool that controls Microsoft Paint through natural language commands. This project enables users to draw shapes based drawings using the available shapes as well as realistic drawings on MS Paint using simple english prompt "Say what to draw not how".

## Prerequisites

Before running this project, ensure you have:

* Python 3.10 or later
* Access to Ollama Cloud models
* Image Generation API key
* Required Python packages from `requirements.txt`


## Tutorial Followed
Use this tutorial to setup your image generation api
* ["FREE AI Image Generation API using CloudFlare" by TrachitZ](https://youtu.be/VliEpQl06pE?si=cp-aSdu91RR_lbwT)
